## Введение

Тип `Decimal` представляет десятичные числа в диапазоне от положительных **79,228,162,514,264,337,593,543,950,335** до отрицательных **79,228,162,514,264,337,593,543,950,335**.

Значение `Decimal` по умолчанию равно `0`. Тип подходит для финансовых расчётов, которые требуют большого количества значимых целых и дробных цифр и отсутствия ошибок округления. Этот тип не устраняет необходимость округления — скорее, сводит к минимуму количество ошибок из-за него.

Когда результат деления и умножения передаётся методу округления, результат не страдает от потери точности.

`Decimal` число — это значение с плавающей точкой, состоящее из знака, числового значения (где каждая цифра находится в диапазоне от 0 до 9) и коэффициента масштабирования, который указывает положение десятичной точки, разделяющей целые и дробные части числового значения.

Двоичное представление `Decimal` состоит из 1-разрядного знака, 96-разрядного целого числа и коэффициента масштабирования, используемого для деления 96-разрядного целого числа. Коэффициент масштабирования неявно равен числу 10, возведённому в степень в диапазоне от 0 до 28. Следовательно, двоичное представление `Decimal` имеет вид:

```
((от -2^96 до 2^96) / 10^(от 0 до 28))
```

где `-(2^96-1)` — минимальное значение, а `2^96-1` — максимальное значение.

Коэффициент масштабирования также может сохранять любые конечные нули в `Decimal`. Эти конечные нули не влияют на значение в арифметических операциях или операциях сравнения.

## Двоичное представление

`Decimal` реализован в виде четырёхэлементного массива 32-разрядных целых чисел со знаком:

```c
typedef struct s21_decimal {
    int bits[4];
} s21_decimal;
```

| Элемент | Содержимое |
|---------|------------|
| `bits[0]` | Младшие 32 бита 96-разрядного целого числа (мантисса) |
| `bits[1]` | Средние 32 бита 96-разрядного целого числа (мантисса) |
| `bits[2]` | Старшие 32 бита 96-разрядного целого числа (мантисса) |
| `bits[3]` | Коэффициент масштабирования и знак |

**Структура `bits[3]`:**

| Биты | Назначение |
|------|------------|
| 0–15 | Не используются, должны быть равны нулю |
| 16–23 | Показатель степени от 0 до 28 (степень 10 для деления мантиссы) |
| 24–30 | Не используются, должны быть равны нулю |
| 31 | Знак числа: `0` — положительное, `1` — отрицательное |

> Битовое представление различает отрицательные и положительные нули. Эти значения считаются эквивалентными во всех операциях.

## Реализованные функции

### Арифметические операторы

| Оператор | Символ | Функция |
|----------|--------|---------|
| Сложение | `+` | `int s21_add(s21_decimal value_1, s21_decimal value_2, s21_decimal *result)` |
| Вычитание | `-` | `int s21_sub(s21_decimal value_1, s21_decimal value_2, s21_decimal *result)` |
| Умножение | `*` | `int s21_mul(s21_decimal value_1, s21_decimal value_2, s21_decimal *result)` |
| Деление | `/` | `int s21_div(s21_decimal value_1, s21_decimal value_2, s21_decimal *result)` |
| Остаток | `mod` | `int s21_mod(s21_decimal value_1, s21_decimal value_2, s21_decimal *result)` |

**Коды возврата:**

| Код | Значение |
|-----|----------|
| `0` | OK |
| `1` | Число слишком велико или равно бесконечности |
| `2` | Число слишком мало или равно отрицательной бесконечности |
| `3` | Деление на ноль |

> При получении чисел, не вмещающихся в мантиссу, используется **банковское округление**.  
> Пример: `79,228,162,514,264,337,593,543,950,335 - 0.6 = 79,228,162,514,264,337,593,543,950,334`

---

### Операторы сравнения

| Оператор | Символ | Функция |
|----------|--------|---------|
| Меньше | `<` | `int s21_is_less(s21_decimal, s21_decimal)` |
| Меньше или равно | `<=` | `int s21_is_less_or_equal(s21_decimal, s21_decimal)` |
| Больше | `>` | `int s21_is_greater(s21_decimal, s21_decimal)` |
| Больше или равно | `>=` | `int s21_is_greater_or_equal(s21_decimal, s21_decimal)` |
| Равно | `==` | `int s21_is_equal(s21_decimal, s21_decimal)` |
| Не равно | `!=` | `int s21_is_not_equal(s21_decimal, s21_decimal)` |

**Коды возврата:**

| Код | Значение |
|-----|----------|
| `0` | FALSE |
| `1` | TRUE |

### Преобразователи

| Преобразование | Функция |
|----------------|---------|
| Из `int` | `int s21_from_int_to_decimal(int src, s21_decimal *dst)` |
| Из `float` | `int s21_from_float_to_decimal(float src, s21_decimal *dst)` |
| В `int` | `int s21_from_decimal_to_int(s21_decimal src, int *dst)` |
| В `float` | `int s21_from_decimal_to_float(s21_decimal src, float *dst)` |

**Коды возврата:**

| Код | Значение |
|-----|----------|
| `0` | OK |
| `1` | Ошибка конвертации |

**Уточнения по `float`:**

- Если число слишком мало (`0 < |x| < 1e-28`) — вернуть ошибку, результат равен `0`
- Если число слишком велико (`|x| > 79,228,162,514,264,337,593,543,950,335`) или равно бесконечности — вернуть ошибку
- При обработке `float` преобразовывать все содержащиеся в нём значимые цифры

**Уточнения по `decimal → int`:**

- Если в числе есть дробная часть — она отбрасывается (например, `0.9` → `0`)

| `0` | OK |
